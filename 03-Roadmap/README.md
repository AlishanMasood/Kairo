# Roadmap

## Purpose

This folder contains planning documents, roadmaps, milestones, and timeline information for the Kairo platform.

## What Belongs Here

- Platform roadmap documents
- Milestone definitions
- Quarterly or phase-based plans
- Prioritization frameworks
- Dependency maps between planned work

## What Does NOT Belong Here

- Completed release documentation (use `07-Releases`)
- Day-to-day task tracking (use external project management tools)
- Architecture decisions (use `08-ADR`)
- Product specifications (use `02-Products`)
