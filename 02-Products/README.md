# Products

## Purpose

This folder contains product specifications, feature definitions, and product-level documentation for all products within the Kairo ecosystem.

## What Belongs Here

- Product specifications
- Feature definitions and requirements
- User stories and use cases
- Product boundaries and scope definitions
- Integration points between products

## What Does NOT Belong Here

- Technical implementation details (use `04-Architecture` or `06-Modules`)
- Release-specific information (use `07-Releases`)
- Roadmap and planning (use `03-Roadmap`)
- Research or prototyping notes (use `09-Research`)
