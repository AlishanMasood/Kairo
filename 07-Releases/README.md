# Releases

## Purpose

This folder contains release notes, changelogs, and version-specific documentation for the Kairo platform.

## What Belongs Here

- Release notes for each version
- Changelogs
- Migration guides between versions
- Known issues per release
- Deprecation notices

## What Does NOT Belong Here

- Future planning (use `03-Roadmap`)
- Architecture decisions that led to changes (use `08-ADR`)
- Development process documentation (use `05-Development`)
- Product specifications (use `02-Products`)
