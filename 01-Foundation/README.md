# Foundation

## Purpose

This folder contains foundational concepts, principles, and definitions that underpin the Kairo platform. These are the baseline references that other documentation builds upon.

## What Belongs Here

- Core platform principles
- Glossary and terminology definitions
- Domain model descriptions
- Foundational constraints and assumptions
- Platform philosophy and guiding values

## What Does NOT Belong Here

- Implementation details (use `04-Architecture` or `06-Modules`)
- Product-specific features (use `02-Products`)
- Temporary research or explorations (use `09-Research`)
- Process documentation (use `05-Development`)
