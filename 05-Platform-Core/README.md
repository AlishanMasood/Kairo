# Platform Core

## Purpose

This folder contains documentation defining the Kairo platform core — the shared infrastructure layer that all products are built upon. It covers platform identity, multi-tenancy, event infrastructure, configuration, and all shared platform services.

## What Belongs Here

- Platform core architecture and definition
- Platform service specifications (Identity, Tenancy, Events, Configuration)
- Shared infrastructure design documents
- Platform capability documentation
- Platform evolution strategy
- Platform service contracts

## What Does NOT Belong Here

- Product-specific architecture (use `04-Architecture` for cross-cutting, product folders for specific)
- Business capability definitions (use `03-Business-Capabilities`)
- Module-level implementation details (use `06-Modules`)
- Development process standards (future `Development` documentation)
