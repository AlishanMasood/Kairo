# Governance

## Purpose

This folder contains governance policies, standards, decision-making processes, and organizational rules that apply across the Kairo platform.

## What Belongs Here

- Decision-making frameworks
- Contribution policies
- Review and approval processes
- Naming conventions and standards
- Documentation standards
- Access control policies
- Compliance requirements

## What Does NOT Belong Here

- Technical architecture decisions (use `08-ADR`)
- Development process documentation (use `05-Development`)
- Product specifications (use `02-Products`)
- Meeting notes or informal discussions
