# Architecture

## Purpose

This folder contains system architecture documentation, technical designs, and structural descriptions of the Kairo platform.

## What Belongs Here

- System architecture overviews
- Component diagrams and descriptions
- Data flow documentation
- Infrastructure architecture
- Integration architecture
- Security architecture
- API design documentation

## What Does NOT Belong Here

- Architecture Decision Records (use `08-ADR`)
- Module-specific implementation details (use `06-Modules`)
- Development process guidelines (use `05-Development`)
- Research or proof-of-concept notes (use `09-Research`)
