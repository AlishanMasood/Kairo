# Templates

## Purpose

This folder contains reusable document templates for creating consistent documentation across the Kairo repository.

## What Belongs Here

- ADR templates
- Product specification templates
- Module documentation templates
- Release note templates
- Research document templates
- Meeting note templates
- Any other recurring document format

## What Does NOT Belong Here

- Actual documentation (use the appropriate numbered folder)
- Code templates or scaffolding (belongs in source repositories)
- Configuration templates (belongs in source repositories)
